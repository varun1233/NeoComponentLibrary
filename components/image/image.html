Images

Images can be responsive to fit the parent's width, and also can be customised to be round or square shaped

Example of Responsive Image

You can add class img-responsive to make your image fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image.

Example of Round & Square Image

You can add class image-round to make your image round shaped.
To make an image square shaped you can add class image-square.
