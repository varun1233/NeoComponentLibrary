Typography

Swift UI uses Sans-serif font as it's default font. The root font-size is set as 10px for this library. Check out below text utilities.

Example of Heading

For heading u can use h1, h2, h3, h4, h5, h6 elements.
The same font-styling is present for class names h1, h2, h3, h4, h5, h6 class. You can add one of these classes to style the text.

Example of Paragraph Text

For paragraph u can use p element.
For different font styles add h1, h2, h3, h4, h5, h6 class to p element.
To align the text add text-align-center, text-align-left, text-align-right class to wrapper element
