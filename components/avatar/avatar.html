Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Example of Image Avatar

Avatar is available in 6 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size avatar-xxl, avatar-xl, avatar-lg, avatar-md avatar-sm avatar-xs (e.g.class="avatar avatar-lg")

Example of Text Avatar

You can use initial letters of user in Avatar as well. You need to include class avatar-text along with avatar and size class. (e.g. class="avatar avatar-text avatar-lg")
